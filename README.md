                                   Hospital Management System

Project Description:
     This project is a Java-based application designed to manage hospital operations, such as patient records and database 
connectivity. It utilizes Object-Oriented Programming principles to ensure a clean and modular structure.

Purpose & Functionality:
   Purpose: To streamline patient data management and provide a reliable interface for hospital staff.It will set the value 
null/dicharge when patient get dicharge..
Main Modules:
   HospitalManagement: Handles the primary logic and user interface.
   PatientEntity: Manages specific patient data structures.
   JdbcTest: Facilitates the connection between the Java application and the MySQL database.

How to Run
   1.Database Setup: Import the `HospitalManagement.sql` file into your MySQL server to set up the necessary tables.
   2.Compile: Compile the source files using `javac *.java`.
   3.Execute: Run the application using `java HospitalManagement`.
   4.Driver:Ensure the `mysql-connector-j-9.7.0.jar` is in your classpath.
